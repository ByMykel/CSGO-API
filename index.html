Introduction

Welcome to the unofficial JSON API for Counter-Strike 2. This API provides access to various data aspects of the game, parsed into JSON format for easier integration and use.

Data are sourced from files maintained at this repository. And you can find this project source code here.

BASE URL
https://bymykel.github.io/CSGO-API/api/{language}

To access information in a specific language, replace {language} in the URL with one of the supported language codes: bg cs da de el en es-ES es-MX fi fr hu it ja ko nl no pl pt-BR pt-PT ro ru sv th tr uk zh-CN zh-TW vi

Object with all items accessible by their id.
